# Spotify Clone (Web)

A modern, high-fidelity Spotify Clone web application. Built with premium styling, smooth transitions, and dynamic audio playback.

## Features (Planned)
- 🎵 **Dynamic Audio Player**: Play, pause, skip, seek, volume control, and shuffle/repeat.
- 🔍 **Search & Discovery**: Find songs, artists, albums, and playlists.
- 📂 **Library & Playlists**: Create, edit, and manage custom playlists.
- 🎨 **Premium UI/UX**: Dark mode by default, glassmorphism elements, hover transitions, and responsive layout.

## Tech Stack
- Frontend: HTML, CSS, JavaScript (React / Next.js / Vite)
- State Management: Context API / Zustand
- Icons: Lucide React / Tailwind (if needed)

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ashmitsingh-ai/CLONE_SPOTIFY_WEB.git
   ```
2. Navigate to the project directory and install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
